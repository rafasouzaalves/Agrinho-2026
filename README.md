# 🌱 Agro Forte, Futuro Sustentável

## 📖 Sobre o Projeto

O projeto **Agro Forte, Futuro Sustentável: equilíbrio entre produção e meio ambiente** foi desenvolvido com o objetivo de conscientizar sobre a importância da agricultura sustentável e demonstrar como a tecnologia pode auxiliar na preservação ambiental sem comprometer a produtividade agrícola.

O site apresenta informações sobre práticas sustentáveis no campo, projetos ambientais e um formulário de contato interativo para engajar os usuários.

---

## 🎯 Objetivos

* Promover a conscientização sobre sustentabilidade no agronegócio.
* Demonstrar a importância do equilíbrio entre produção agrícola e preservação ambiental.
* Aplicar conhecimentos de HTML, CSS e JavaScript em um projeto completo e funcional.
* Desenvolver uma interface moderna, responsiva e acessível.

---

## 💻 Tecnologias Utilizadas

### HTML5

* Estrutura semântica do site.
* Organização do conteúdo em seções.
* Formulário de contato.

### CSS3

* Estilização da interface.
* Layout responsivo utilizando Media Queries.
* Efeitos visuais e transições.
* Centralização e organização dos elementos.

### JavaScript

* Manipulação do DOM.
* Validação dinâmica do formulário.
* Exibição de mensagens de feedback.
* Rolagem suave entre seções.
* Controle de eventos de clique.

## ⚙️ Funcionalidades

### 🌿 Página Inicial

Apresenta o tema do projeto e uma breve introdução sobre sustentabilidade no agronegócio.

### 📖 Seção Sobre

Explica os objetivos e a importância da agricultura sustentável.

### 🚜 Projetos Sustentáveis

Exibe exemplos de iniciativas que contribuem para a preservação ambiental:

* Plantio Inteligente
* Energia Renovável
* Reciclagem de Resíduos

### 📩 Formulário de Contato

Possui validação em JavaScript para garantir que todos os campos sejam preenchidos antes do envio.

### 📱 Responsividade

O site adapta automaticamente seu layout para celulares, tablets e computadores.

---

## 🧠 Lógica Implementada em JavaScript

### Rolagem Suave

Quando o usuário clica no botão "Saiba Mais", a página realiza uma navegação suave até a seção "Sobre".

```javascript
scrollIntoView({
    behavior: "smooth"
});
```

### Validação de Formulário

O sistema verifica se todos os campos foram preenchidos antes de permitir o envio.

```javascript
if(nome === "" || email === "" || mensagem === ""){
    // Exibe mensagem de erro
}
```

### Manipulação do DOM

Os elementos da página são acessados e modificados dinamicamente através do JavaScript.
